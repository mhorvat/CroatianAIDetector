# CroatianAIDetector
This repository contains the complete codebase created for the research presented in the scientific paper "Detecting AI-Generated Text in the Croatian Language Through Syntactic, Semantic, and Stylistic Analysis"

The developed application is available only for research and non-commercial use.

For more information and instructions on how to use the appliation, contact the application's creator *Ariana Dobrostal* (ariana.dobrostal@proton.me).
