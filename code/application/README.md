EN

Starting the application
- Python version: Python 3.10.4
- installation of required libraries: pip install -r requirements.txt
- running the application: python A10A.py

Using the application
1. Paste the text content or the path to the text into the text area.
2. Fill in the other data if known (optional):
- Title – title or topic of the text
- Generated - check the box if it is known that the text was generated
- Generated times – the number of times the generation query was edited or supplemented
- Model – the model with which the text was generated
- Query – the query used in text generation
- Path - check the box if the text area contains the path to the text
3. Press the ‘Detect’ button.

Pokretanje aplikacije
- Python verzija: Python 3.10.4
- instalacija potrebnih knjižnica: pip install -r requirements.txt
- pokretanje aplikacije: python A10A.py

Korištenje aplikacije
1. Zalijepiti sadržaj teksta ili putanju do teksta u prostor za tekst
2. Ispuniti ostale podatke ako su poznati (opcionalno):
- Naslov - naslov ili tema teksta
- Generirano - označiti kvačicu ako je poznati da je tekst generiran
- Generirano puta - broj puta koliko je upit za generiranje uređivan ili nadopunjavan
- Model - model kojim je tekst generiran
- Upit - korišteni upit u generiranju teksta
- Putanja - označiti kvačicu ako je u prostoru za tekst putanja do teksta
3. Pritisnuti gumb 'Detektiraj'
